Privacy Policy — pdf-sign-extension.pages.dev

Last updated: April 22, 2026

This policy covers the marketing website at pdf-sign-extension.pages.dev. For the PDF Sign Chrome extension itself, see the extension privacy policy.

What this website does

This is a static marketing page describing the PDF Sign Chrome extension. It contains no forms, no accounts, and no document handling — all of that lives inside the Chrome extension, not here.

Data we collect on this website

Aggregated analytics

We use Cloudflare Web Analytics to count page views and identify which external sites link to us. It is cookie-free and does not track individual users across sessions. Cloudflare processes your IP address and user agent to produce aggregated counts (country, browser, referrer) but does not store individual-level records on our side.

Data we do NOT collect on this website

  • We set no cookies
  • We host no forms
  • We use no tracking pixels, fingerprinting, or retargeting
  • The site has no sign-in and no account system — those live in the Chrome extension

Third-party services

  • Cloudflare (Pages + Web Analytics) — hosts the website and provides anonymous analytics. Cloudflare sees your IP address (required to deliver the page) and processes it under its own terms. Subject to Cloudflare's Privacy Policy.
  • unpkg.com (JS CDN) — when you load the page, your browser fetches a small JavaScript library (Lenis smooth-scroll) from unpkg.com. That request exposes your IP address and user agent to unpkg. It is used only to deliver the file and is not used for analytics or identification. unpkg runs on top of Cloudflare; subject to Cloudflare's Privacy Policy.
